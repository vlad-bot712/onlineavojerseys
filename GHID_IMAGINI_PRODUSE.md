# GHID IMAGINI PRODUSE - AVO JERSEYS

## Locația fișierelor
Toate imaginile pentru produse se află în:
```
/app/frontend/public/images/products/
```

## Convenția de denumire
```
[echipa]-[an]-[kit].jpg
```

Unde:
- `echipa` = numele echipei cu litere mici, fără diacritice, spații înlocuite cu cratimă
- `an` = anul (2024, 2025, 2026)
- `kit` = tipul kit-ului (first, second, third)

## CUM SĂ ADAUGI IMAGINI NOI

1. Redenumește imaginea ta conform convenției de mai sus
2. Copiază imaginea în folderul `/app/frontend/public/images/products/`
3. Imaginea va fi afișată automat în site

## NOTĂ IMPORTANTĂ
- Dimensiuni recomandate: 800x800 pixeli sau proporții pătrate
- Format: JPG
- Calitate: medie-înaltă pentru încărcare rapidă
